GrowPi

The GrowPi, is a Raspberry Pi 3B+, with a grovePi+. It has moisture, temperature and humidity sensors.
Every interval(10 mins):

  • It reads all sensor data
  • Appends the data to a csv file
  • Uploads the CSV Data to this site
  • Takes a picture, with the picamera, when lights are on and uploads it to this site
